# iris-admin

* openFx기반으로 배포 된 IRIS 관련 OpenFx 함수들을 사용할 수 있는 테스트 페이지입니다.

---

## 1. before settings
### install on server
`npm`  
`npx`  

### for react packages
`npm install gh-pages --save-dev`  
`npm install react-router-dom`  
`npm install cross-env --dev`  

## 2. build & deploy
### initial npm install
```
$ npm install
```
### localhost deploy
```
$ npm start
```
### gitblog deploy
```
$ npm run deploy
```
